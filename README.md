# 📌 Student Appointment Booking System

## 📖 Description

The **Student Appointment Booking System** is a mini web-based prototype designed to allow students to book appointments with school staff online. The system aims to reduce long queues and improve scheduling efficiency by providing a simple and user-friendly interface.

---

## 🚀 Features

### ✅ Functional Requirements

* User login authentication
* Book an appointment
* View scheduled appointments
* Admin approval/rejection (conceptual)
* Logout functionality

### ⚙️ Non-Functional Requirements

* Fast loading (within 3 seconds)
* Responsive design (mobile & desktop)
* Secure data handling
* User-friendly interface
* Supports multiple users

---

## 🖥️ System Pages

* **Homepage** – Introduction and navigation
* **Login Page** – User authentication
* **Dashboard** – Main user interface after login
